World Radio · TUI Design

v0.4 draft · 23 May 2026 · for Ratatui

Терміналу не потрібні рамки. Сім ретро-, кібер-, атомпанк- та mainframe-палітр (+ окрема секція edge-станів і DSP-панелі), всі чотири MVP-екрани в кожній, плюс hero-візуалізацій: vertical bars, analog VU needle, horizontal tuner dial, waterfall spectrogram, cyber spectrum scope, diagnostic stat readout, signal history log. Текст у блоках <pre> — чистий ASCII/Unicode (емоджі-прапори + ISO-код), його можна скопіювати і вставити в код. Внизу — палітри в hex, 256/16-color фолбек і мапінг на віджети Ratatui.

Ratatui widget mapping

Header bar — Paragraph with two Lines (logo+title left, status+clock right). No border.

Tabs row — Tabs widget with custom divider " · " and highlight_style reversed (the inv token).

Station list — List with ListItem::new(Line::from(vec![Span])). Selected = ▸ prefix + bold; normal = · prefix; disabled = all dim + trailing ✗ dead.

Search input — Paragraph with manual cursor (▌) at end. Set Frame::set_cursor for OS-blinking variant.

Filter checkboxes — List with [✓] / [ ] prefix. Active group header in fg, inactive in dim.

Spectrum bars (V1) — BarChart with bar_set::THREE_LEVELS or a custom Symbols::Bar. Color: gradient via per-bar style. Hint: for height-based color use a custom render — split bars into bottom (dim), mid (accent), top (hot).

VU meter (V2) — Canvas widget with Line shape. Compute needle angle from RMS; draw arc tick-marks with · and the over-zero region with h tokens. Needle endpoint = ●.

Tuner dial (V3) — Gauge repurposed: ratio = (freq − min) / (max − min). Render ──────●────── with ▼ marker above. Or Sparkline reversed.

Level bars — LineGauge per channel with line_set::THICK. Use eighths blocks for sub-cell precision.

Status bar — Paragraph pinned to last row, key=accent, label=dim, separated by " ".

Layout · minimum size

  • Min: 80 × 24 (POSIX classic). Hides filter panel on Browse, collapses VU to single-channel.
  • Comfortable: 110 × 32 — design above is calibrated for this.
  • Wide: 140+ — spectrum gets full 20Hz–20kHz range, Now Playing splits into 2 columns (info | meter).
  • Use Layout::vertical([Length(3), Length(1), Min(0), Length(1)]) for: header → tabs → body → status.

State patterns — for engineering

Edge-case screens вище показані в Amber CRT як референс. Структура та класи однакові для всіх 7 тем — палітра підставляється з обраної теми. Декілька правил:

  • Loading splash blocks input except Q. Cancel returns OS exit code 130 (SIGINT). Progress is best-effort — show what you've got.
  • Empty search never crashes the layout — empty list + helpful suggestions in same panel. Show which active filters would expand results (drop hints).
  • Empty favorites is a quick-start, not an error. Curated suggestions act as "press F here" tutorial.
  • Offline is the only state that takes over the whole screen and dims tabs. Auto-retry with exponential backoff (1s → 2s → 4s → 8s → 15s, then steady).
  • Buffering stays in Now Playing layout — header, tabs, status all visible — but body replaced with stack-of-bars showing fill-over-time. After 5s grace, show "skip to next favorite" hint.
  • Stream dead shows a short failure log (last 24h, 3 entries max). Most useful action is jump to next favorite — make it the default with ▸.
  • Equalizer is a modal overlay invoked with E from any screen. Implement as full-screen Paragraph + BarChart for the EQ curve. Apply via the audio output sink (cpal/rodio).
